# Glyxon-PRIMALS-VOICE-PCR-
GLYXON PRIMALS: A brutalist, open-source qPCR thermal cycler cast in concrete. Featuring photonic IR heating, a cordless galvanic lid, and PJ—a local voice assistant for hands-free DNA amplification. Integrated with Matter &amp; Open Home, it merges street art with high-fidelity genetics. The art of technique, the power of precision.
